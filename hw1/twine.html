Twine is a text adventure open source tool. Twine is very beginner friendly and can use as littel as no coding if the user aims to make a simple story with diverging paths. Twine uses HTML to display the story , this makes the work done on it feasible to read on any device that can use a web browser. Twine has different versions that can be used to change the format the story is displayed in. Usually Twine is used for text adventure but with code can expand to make small video games or be used to make diverging paths in movies. Normally it is used to make text adventures.

Twine uses HTML, CSS and JS
